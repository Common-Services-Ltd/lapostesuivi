La Poste Suivi

À propos

La Poste Suivi est une solution de suivi professionnelle en back-office utilisant l’API Suivi de La Poste.
Suivez vos colis La Poste, Colissimo et Chronopost avec un seul module devient possible, et même changer de statut automtiquement en fonction du suivi !

Fonctionnalités

La nouvelle API La Poste vous permettra de suivre aussi bien vos colis La Poste, Colissimo et Chronopost de façon simple et automatisé grâce à une interface claire et sans fioriture.
Il est également possible de changer le statut des commandes automatiquement en fonction des statuts de suivi des colis.

Il suffit de souscrire à l’API sur https://developer.laposte.fr/products/suivi/latest pour recevoir votre clé API.

Le service est fiable et pérènne, vous utilisez une API officelle La Poste ce qui vous garanti le bon fonctionnement du service.

Les “+” pour vous

  • Module de suivi simple et efficace (~2 minutes pour la configuration)
  • Mise a jour automatique des suivis et statuts de vos commandes
  • Le suivi est affiché au marchand dans un nouvel onglet dédié sur la fiche de commande
  • Le suivi est affiché au client (front-office) dans l’historique des commandes

Les “+” pour le client

  • L’assurance de savoir à tout moment l’avancement de la livraison, la position du colis
  • Une information claire et pertinente

Prérequis

  • Technique :
    • Librairie cURL installée
    • Version PHP supèrieure à 5.3.0 (idéalement 5.6+)
  • API :
    • S’inscrire et créer un projet pour l’API Suivi sur https://developer.laposte.fr/products/suivi/latest
